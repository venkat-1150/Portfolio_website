# E. Venkat Reddy — Personal Portfolio Website

A clean, responsive personal portfolio website built with pure **HTML** and **CSS** — no frameworks, no dependencies.

---

## 🌐 Live Preview

> Open `index.html` in any browser to view the portfolio locally.

---

## 📁 Project Structure

```
portfolio/
├── index.html        # Main portfolio file
├── meeee.jpeg        # Profile photo
├── Qubitech.jpg      # Quantum Fundamentals certificate image
└── README.md         # This file
```

---

## ✨ Features

- **Hero Section** — Profile photo with animated rotating ring, name, and social links
- **About** — Bio, stats (CGPA, Projects, Certs), and skill tags
- **Education** — Academic history with scores and years
- **Projects** — Cards with background images, tech stack tags, and hover effects
- **Certifications** — 6 certificate cards with background images and descriptions
- **Volunteering** — Community involvement
- **Contact** — Contact info + Python-themed code window
- **Smooth scroll** and **fade-in animations** on scroll
- Fully **responsive** for mobile devices

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties, Animations)
- Vanilla JavaScript (Intersection Observer for scroll animations)
- Google Fonts — Playfair Display, DM Sans, Space Mono

---

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/venkat-1150/portfolio.git
   ```

2. Navigate into the folder:
   ```bash
   cd portfolio
   ```

3. Open `index.html` in your browser — no server needed!
